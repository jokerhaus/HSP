# Release Artifacts

The v1.0 release set publishes:

- protocol specification
- CDDL schemas
- registries
- golden vectors
- Rust reference server (`hspd`)
- Rust HTTP/3 gateway (`hsp-gw`)
- Rust S3-like distribution service (`hsp-s3`)
- Rust CDN distribution service (`hsp-cdn`)
- Rust conformance harness (`hsp-conformance`)
- Go SDK (`hsp-go`)
- Go CLI (`hspctl`)

Mandatory release evidence and security artifacts:

- SPDX JSON SBOM generated by Syft
- dependency-review PR gate output
- `cargo audit` report
- `govulncheck` reports for:
  - `sdk/go`
  - `cli/hspctl`
- `hsp-conformance` JSON report with compatibility and negative corpus results
- release-evidence bundle:
  - `docs/release-evidence/security-gates.md`
  - `docs/release-evidence/runbook-drills.md`
  - `docs/release-evidence/compatibility-matrix.md`
  - `docs/release-evidence/external-review.md`

Operational documents included in the release set:

- incident response runbook
- key recovery and rewrap runbook
- replication lag handling runbook
- CDN purge failure handling runbook
