## Reproducible Bundle

Generate a fresh local review bundle with:

```bash
./scripts/generate_release_review_bundle.sh
```

This writes a timestamped artifact set under `artifacts/release-review/` and
updates the `latest` symlink for handoff to operators or external reviewers.
The script no longer downloads Syft during execution; install a pinned Syft
release in the CI image or operator workstation first.

## Notes

- `cargo audit` is pinned to a clean git configuration in CI and release
  commands because workstation-level git credential rewrites can interfere with
  advisory DB fetches.
- CI also publishes:
  - dependency-review output on pull requests
  - SPDX JSON SBOM artifact generated by Syft
  - `cargo audit`, `govulncheck`, and conformance report artifacts
- CI GitHub Actions are pinned to immutable commit SHAs. Update those SHAs
  deliberately as part of dependency governance, not implicitly through mutable
  tags.
